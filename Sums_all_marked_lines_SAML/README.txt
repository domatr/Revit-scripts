"Sums all marked (2D or 3D) lines in type and total length."

Date                = 09.MAY.2024
Python              = 3.0-3.6
Revit               = 2023
IronPython          = 3.4.1 (3.4.1.1000)

Description:
Analyzes all selected lines, calculates their respective lengths,
Totals for each type and the total of all selected lines.
_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_
How-to:
-> Select all the lines you want.
-> Run the script.
-> The calculation results are displayed in the terminal.
